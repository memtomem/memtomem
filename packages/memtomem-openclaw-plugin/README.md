# memtomem — OpenClaw Plugin

OpenClaw plugin that bridges the memtomem MCP server into the OpenClaw AI
gateway, exposing memory tools over stdio.

Detailed architecture, configuration, and usage notes live in the private
`memtomem/memtomem-docs` repo (`packages-archived/openclaw-plugin.md`).

For end-user installation see the public
[memtomem README](https://github.com/memtomem/memtomem).
